// job-tracker

Workflow

  1. Run python jobs/filter.py to get today's H-1B matches from GitHub repos
  2. Paste a job URL into Claude Code → type /career-ops → get A–F score + tailored CV PDF
  3. Add promising jobs here (score B+ recommended) and track status through the pipeline
  4. For skills gaps found by career-ops, run /career-ops training to get course recommendations
  5. Before interviews, run /career-ops deep [company] for company research + prep questions
