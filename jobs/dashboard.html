Workflow

  1. Click ↻ Load Latest to fetch today's GitHub Actions job results automatically

  4. Paste a job URL into Claude Code → /career-ops → get A–F score + tailored CV PDF
  5. For skills gaps, run /career-ops training; for interview prep, run /career-ops deep [Company]
